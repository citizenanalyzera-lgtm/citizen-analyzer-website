1. Installation

Chrome / Edge / Brave / Opera

  • Rendez-vous sur le Chrome Web Store (lien bientôt disponible)
  • Cliquez sur "Ajouter à Chrome"
  • L'icône Citizen Analyzer apparaît dans votre barre d'extensions

Firefox

  • Rendez-vous sur Firefox Add-ons (lien bientôt disponible)
  • Cliquez sur "Ajouter à Firefox"
  • Autorisez l'extension quand Firefox vous le demande

Installation manuelle (développeurs)

  • Téléchargez le code source depuis GitHub
  • Dans Chrome, allez dans chrome://extensions/
  • Activez le "Mode développeur" en haut à droite
  • Cliquez "Charger l'extension non empaquetée" et sélectionnez le dossier extension/
🚀

2. Première analyse

  • Ouvrez un article — N'importe quel article de presse, discours retranscrit, programme politique ou texte de loi.
  • Cliquez sur l'icône — L'extension s'ouvre dans un panneau latéral à droite de la page.
  • Lancez l'analyse — Cliquez sur "Analyser cette page". L'analyse prend environ 2 à 5 minutes selon la longueur du texte.
  • Consultez les résultats — Le panneau affiche un résumé rapide (Niveau 1), puis vous pouvez approfondir chaque dimension.
📖

3. Lire les résultats

L'analyse est organisée en niveaux progressifs de profondeur :

N1
Résumé citoyen

Score global, mécanismes rhétoriques principaux, groupes affectés, indicateur de régime. Accessible à tous en un coup d'œil.

N2
Analyse détaillée

Tiroirs thématiques : rhétorique complète, analyse de régime, impact distributionnel, vérification factuelle.

N3
Fiabilité

Évaluation de la fiabilité de l'analyse elle-même : sources vérifiées, limites identifiées, niveau de confiance.

N4
Exploration

Posez vos propres questions sur le texte analysé. L'IA répond en s'appuyant sur l'analyse déjà effectuée.

N5
Et moi dans tout ça ?

Impact personnalisé selon votre profil socio-économique. Confidentiel — aucune donnée n'est envoyée à un serveur.

📋

4. Mode programme politique

Citizen Analyzer dispose d'un mode spécifique pour les programmes politiques, utilisant le framework MARPOR (56 catégories thématiques). Ce mode ajoute des analyses spécifiques :

  • Identité programmatique — Positionnement sur les 7 domaines MARPOR
  • Couverture thématique — Quels sujets sont traités, quels sont les angles morts
  • Cohérence budgétaire — Les mesures annoncées sont-elles financièrement cohérentes
  • Crédibilité — Les promesses sont-elles réalistes au vu du contexte

Le mode programme est accessible via le sélecteur en haut du module Import, ou automatiquement lorsque l'extension détecte un programme politique.

📄

5. Importer un document

Vous pouvez analyser des documents qui ne sont pas sur une page web :

  • PDF — Glissez un fichier PDF dans la zone d'import. L'extension extrait le texte automatiquement.
  • Texte libre — Collez directement un texte (minimum 2 000 caractères) dans la zone de saisie.

Après l'import, choisissez entre "Analyser comme discours" ou "Analyser comme programme" pour lancer l'analyse avec le mode approprié.

❓

6. Questions fréquentes

  • Combien de temps dure une analyse ? Environ 2 à 5 minutes, selon la longueur du texte et la charge du serveur.
  • L'extension fonctionne-t-elle en local ? Non, l'analyse nécessite un appel au serveur backend qui utilise un modèle de langage.
  • Mes données sont-elles collectées ? Non. Le texte est envoyé au serveur uniquement pour l'analyse, puis supprimé. Aucune donnée personnelle n'est stockée.
  • L'outil est-il neutre politiquement ? Oui. Les mêmes grilles d'analyse sont appliquées à tous les bords politiques. Consultez notre page Méthodologie pour plus de détails.
  • Quelles langues sont supportées ? L'interface est disponible en 6 langues : français, anglais, allemand, espagnol, italien et portugais. L'analyse fonctionne sur des textes dans ces mêmes langues.
  • Je trouve un bug, que faire ? Envoyez-nous un message à contact@citizen-analyzer.org ou ouvrez une issue sur GitHub.
